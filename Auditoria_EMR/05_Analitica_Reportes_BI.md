# Herramientas y Módulos de Analítica (Reportes, BI, Dashboards)
[[ _TOC_ ]]

Fecha: 12/11/2025

## BI y dashboards
- Power BI Embedded en la aplicación web para KPIs clínicos y administrativos.
- Gráficos en UI con Kendo UI Charts y/o Chart.js.

## Reportería
- >1.300 reportes RDLC (LocalReport) integrados con ReportViewer.
- Exportación a PDF y Excel con EPPlus/ClosedXML/Select.Pdf.

## Módulos analíticos
- Servicios Windows para información gerencial y financiera.
- Esquemas de datos epidemiológicos (Epide) y calidad clínica.

## Métricas ejemplo
- Disponibilidad clínica: tiempos de atención, ocupación de camas, productividad quirúrgica.
- Administrativas: órdenes/día, autorizaciones, facturación y RIPS.
