# Auditoría GHIPS ERP — Carpeta de Entrega

Este directorio contiene el documento general consolidado para auditoría de GHIPS ERP, compilado a partir de la documentación existente en los módulos Asistencial (EMR), FevRips, Financiero, PQRS y Activos.

- Marca: GHIPS
- Desarrollador: TEAM GHIPS
- Fecha: 10/11/2025

## Contenido

- `GHIPS_ERP_Auditoria_General.md` — Documento principal con todas las secciones solicitadas por la auditoría: arquitectura, componentes EMR, documentación técnica, frameworks, modelo de datos, analítica/BI, métricas, ambientes, interoperabilidad (incl. HL7/FHIR), seguridad/privacidad, roles, clasificación de información, escalabilidad, pruebas de carga, balanceo, failover y procedimientos de monitoreo/alertamiento.

## Fuentes utilizadas

Se han referenciado y sintetizado los siguientes archivos del repositorio:
- Asistencial (EMR): Arquitectura general, Interoperabilidad, Seguridad/Privacidad, Escalabilidad/Desempeño.
- FevRips: Arquitectura, Modelo de datos, Seguridad y privacidad.
- Financiero: Arquitectura y componentes, Reportes y Analítica, Recomendaciones de modernización.
- PQRS: Diagramas de arquitectura, Documento de auditoría del frontend.
- Activos: Arquitectura, Modelo de datos.

## Cómo mantener este entregable

1. Actualice primero los documentos fuente en cada módulo.
2. Sincronice referencias y métricas en `GHIPS_ERP_Auditoria_General.md`.
3. Mantenga la fecha y versión en el encabezado.
4. Verifique enlaces relativos y nombres de archivo antes de entregar.
